AI8 Smart Parcel Box

Secure IoT-Based Delivery Solution

About the Project

The AI8 Smart Parcel Box is designed to solve the issue of missed or insecure deliveries. It integrates Raspberry Pi 4B, motion sensors, dual cameras, cloud storage, and a custom mobile app for secure and remote parcel management.

Features

  • Motion-triggered video and image capture
  • Secure media storage on Google Cloud Platform
  • Remote parcel box unlocking via mobile app
  • Telegram alerts for delivery notifications
  • Future enhancements: Two-way voice communication, facial recognition

How It Works

When motion is detected, the system captures images and video, stores them securely in the cloud, and sends a notification to the user. Users can monitor deliveries, unlock the parcel box remotely, and review media from their smartphone.
